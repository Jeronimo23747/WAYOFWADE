## Git Workflow Instructions
1. **Cloning the Repository**  
   Use the following command to clone the repository to your local machine:
   ```bash
   git clone https://github.com/Jeronimo23747/WAYOFWADE.git
   ```

2. **Creating a Branch**  
   Before making changes, create a new branch:
   ```bash
   git checkout -b your-feature-branch
   ```

3. **Making Changes**  
   Make the necessary changes in your local repository.

4. **Committing Changes**  
   After making your changes, commit them with a meaningful message:
   ```bash
   git commit -m "Description of changes"
   ```

5. **Pushing Changes**  
   Push your changes to the remote repository:
   ```bash
   git push origin your-feature-branch
   ```

6. **Creating a Pull Request**  
   Go to the repository on GitHub and create a pull request from your branch to the main branch.
